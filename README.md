# QuantraVision

AI-powered pattern detection for traders. 100% offline, privacy-first Android app.

## Features
- 109 chart patterns detected in real-time
- Hybrid AI detection (OpenCV + TensorFlow Lite)
- Freemium pricing: Standard ($9.99) and Pro ($24.99)

```bash
./gradlew assembleDebug
```
